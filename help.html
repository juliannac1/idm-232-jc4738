Help Page

Getting Started

Browsing Recipes

  • You can explore recipes by scrolling through the homepage or visiting the Recipes section. Click on any recipe card to view full details.

Using Filters

  • Filter recipes by category such as:
  • No Meat
  • No Eggs
  • Dairy Free

Filters help you find recipes that match your preferences.

Using Search

Use the search bar to quickly find recipes by:

  • Name (e.g., “Chicken Adobo”)
  • Ingredient (e.g., “garlic”)
  • Type (e.g., “stir fry”)

Viewing a Recipe

Each recipe page includes:

  • Ingredients list
  • Step-by-step instructions
  • Prep and cook time
  • Serving size
